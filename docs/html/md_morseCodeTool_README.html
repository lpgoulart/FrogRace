morseCodeTool

Algorithm to Translate any message from Morse code to Text and write vice versa in a local file, writen in C++ on Programming Language 1 grade at the IT Bachelor degree - UFRN

Installation

To install the latest release:

get the source code from GitLab repository:

1 $ git clone http://forcohen@projetos.imd.ufrn.br/forcohen/morseCodeTool.git

simple, use the Makefile on root folder:

1 $ make

Finally, you can use the binary file on bin directory.

As of version 1, morseCodeTool requires GCC for compiling the source codes (MinGW for MS system).

Getting Started

Run: 

cd bin
./exec

this should give you a output menu with option: 

Welcome to the Morse code translator tool

Here you can converte any text to a Morse code message
and go the back way to.

(1)Morse to Text
(2)Text to Morse
(0)Exit
Select a valid function:

Choose one, then enter a file name or a string for the condig/decondig process. 

> 2
Enter your message below:
> a simple text

-------------
your translated message has been writed in myMessage.txt on the src directory

Note that your input file has to be in src directory!
